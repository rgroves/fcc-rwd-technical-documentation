Positional Notation

The numeral systems discussed herein use a positional notation or can be said to be positional numeral systems, or just positional systems in the context of this document. A positional system is a system for representing numbers using an ordered set of numeral symbols (or digits), in which each digit has a value that depends on its position within a number.

For any number expressed using a modern positional system, you will need to know the base (or radix) of the system to determine the value that each individual digit contributes to the total value of the number. The base is the number of unique digits used by the system. The letter b is the symbol typically used as a notation for base. A base can also be indicated by using a phrase like base-b (where b is the base number, e.g. base-2, base-10, etc.).

For example, if we used the entire English alphabet as the digits of a positional system, that system would be a base-26 system (because the English alphabet is a set of 26 letters) and every digit would assigned a value from 0 to 25. Note that the numeral symbols used do not have to be decimal digits. Keep this concept of using letters as digits in mind, as we will revisit it when we discuss the hexadecimal numeral system.

Once the base is known, you can use it to determine the value of each position (or place) in a number. Each position has a value assigned to it, expressed in powers of the base (bx). We'll use the following to denote the positions of a number, from left-to-right:

  • pN
  • ...
  • p2
  • p1
  • p0

Meaning the right-most digit is at postition p0, the next digit to the left is at postition p1, and so forth. With that we can express the value given to each position as follows. The value of:

  • pN is bN
  • ...
  • p2 is b2
  • p1 is b1
  • p0 is b0

Now, to determine each digit's individual value, you take the value of the digit and multiply it by the value of the position where the digit is located in the number. By summing each digit's individual value, you get a number's total value.

In summary, to get the total value (t) for a number, you take the value of the digit (d) at each position (pN) and multiply it by the value of that position. In the notation below we express the value of the digit at position pN as dpN

t = sum( (dpN * bN) + ... + (dp2 * b2) + (dp1 * b1) + (dp0 * b0) )

Concepts in Practice: Devising a Base-4 Numeral System

Let's explore the concepts introduced above and see how we could devise our own base-4 numeral system. For simplicity's sake we are only going to allow positive whole number values to be represented in our system, so we will not have to deal with any negative or fractional values. We will also just show a method of converting our base-4 numbers to their decimal equivalents using JavaScript.

First, we will determine the symbols that will be used to represent our digits. We can use the letters A, B, C, and D. Then we will give each symbol a value, 0, 1, 2, and 3 respectively.

It should be noted, we could just use the symbols '0', '1', '2', and '3' for this system, but this shows that the digits used in the system are independent of the system itself. In fact, we could have used any letters or symbols available to us on a keyboard as our symbols for representing the values from 0 to 3. We shall keep it simple though and stick with A - D.

function convertBase4StrToDecimal(str) {
  let total = 0;

  // Using an object to represent our digits
  // and their values, with the object properties
  // representing each symbol used for a digit and
  // the object property values representing the
  // digit's value.
  const digits = {
    A: 0, // A = 0
    B: 1, // B = 1
    C: 2, // c = 2
    D: 3, // D = 3
  };

  for (let power = 0; power < str.length; power++) {
    // Get the index of the current digit's position
    // starting from the right-side.
    let index = str.length - (power + 1);

    // Get the digit at that position.
    let digit = str[index];

    // Get the value of that digit
    let value = digits[digit];

    // If the digit is not a valid symbol in our
    // system, then return NaN.
    if (value === NaN) {
      return value;
    }

    // Keep a tally of the product of each digit's
    // value and a power of 4 based on the digit's
    // position. (e.g. ones position: 4^0; fours 
    // position: 4^1, sixteens position: 4^2, etc.)
    total += value * Math.pow(4, power);
  }

  return total;
}
